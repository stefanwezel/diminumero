Aprender os Números em Espanhol

Esta página ensina os padrões principais dos números em espanhol — com muitos exemplos (e as armadilhas comuns: cien vs. ciento, acentos, veintiún, un millón).

0–29: elementos fundamentais

O melhor é memorizar 0–15 (cero … quince). A partir daí surgem os padrões:

  • 16–19: dieci- + número → dieciséis, diecisiete, dieciocho, diecinueve
  • 20: veinte
  • 21–29: veinti- + número → veintiuno, veintidós, veintitrés, …
{{ get_text('learn_quick_check') }}: 18, 22, 29?

18 dieciocho · 22 veintidós · 29 veintinueve

30–99: dezenas + "y" + unidades

A partir de 30, o padrão padrão é: dezenas + (y + unidades) quando o algarismo das unidades não é zero.

  • 30 = treinta
  • 31 = treinta y uno
  • 42 = cuarenta y dos
  • 98 = noventa y ocho

100–999: cien vs. ciento e centenas

  • 100 = cien
  • 101–199 = ciento + … (ex.: 115 = ciento quince)
  • 200–900: doscientos, trescientos, cuatrocientos, …

Centenas irregulares frequentemente esquecidas: quinientos (500), setecientos (700), novecientos (900).

{{ get_text('learn_quick_check') }}: 100, 115, 500, 742?

100 cien · 115 ciento quince · 500 quinientos · 742 setecientos cuarenta y dos

Milhares

  • 1000 = mil (não "un mil")
  • 2000 = dos mil
  • 4321 = cuatro mil trescientos veintiuno

Antes de mil, uno → un (apócope): 21.000 = veintiún mil, 31.000 = treinta y un mil. (Com um substantivo feminino também se pode ver "treinta y una mil personas", mas para "números sozinhos", "treinta y un mil" é o padrão.)

Milhões e números grandes

  • 1.000.000 = un millón
  • 2.000.000 = dos millones
  • 1.234.567 = un millón doscientos treinta y cuatro mil quinientos sesenta y siete
  • 1.000.000.000 = mil millones (também: "un millardo")
  • 1.000.000.000.000 = un billón (escala longa espanhola)

Ortografia e acentos

Regra de ortografia: 16–19, 21–29 e todas as centenas são normalmente escritos como uma única palavra (dieciséis, veintidós, doscientos). Formas como "veinte y uno" são consideradas arcaicas.

Acentos que importam muito: dieciséis, veintidós, veintitrés, veintiséis.

Erros comuns

  1. cien vs. ciento: 100 = cien, mas 101 = ciento uno.
  2. 21.000: não "veintiuno mil" → veintiún mil.
  3. 1.000.000: não "uno millón" → un millón.
  4. Acentos em falta: especialmente 16/22/23/26.
  5. Centenas irregulares: quinientos / setecientos / novecientos.
