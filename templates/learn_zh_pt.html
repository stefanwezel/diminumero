Aprender os Números em Chinês

Esta página cobre os números em chinês mandarim — um sistema posicional regular com dígitos e caracteres de valor posicional. O principal desafio é saber quando usar 零 (zero) como marcador de lacuna.

Dígitos: 0–9

  • 0 = 零 (líng)
  • 1 = 一 (yī)
  • 2 = 二 (èr)
  • 3 = 三 (sān)
  • 4 = 四 (sì)
  • 5 = 五 (wǔ)
  • 6 = 六 (liù)
  • 7 = 七 (qī)
  • 8 = 八 (bā)
  • 9 = 九 (jiǔ)

10–99: dezenas

十 (shí) significa 10. O padrão é dígito + 十 para múltiplos de dez, e 十 + dígito para os números de 11 a 19:

  • 10 = 十
  • 11 = 十一
  • 20 = 二十
  • 25 = 二十五
  • 99 = 九十九

Centenas

百 (bǎi) significa 100:

  • 100 = 一百
  • 200 = 二百
  • 350 = 三百五十
  • 101 = 一百零一 (零 marca a lacuna)

Milhares

千 (qiān) significa 1.000:

  • 1.000 = 一千
  • 2.000 = 二千
  • 5.678 = 五千六百七十八
  • 1.001 = 一千零一

Números grandes: 万 (10.000+)

O chinês agrupa os números em unidades de 10.000 (万, wàn):

  • 10.000 = 一万
  • 100.000 = 十万
  • 1.000.000 = 百万

As regras do zero (零)

零 é inserido como marcador quando há uma lacuna entre os valores posicionais:

  1. 101 = 一百零一 (lacuna onde ficaria a dezena)
  2. 1.001 = 一千零一 (lacuna onde ficariam a centena e a dezena)
  3. 1.010 = 一千零一十

Vários zeros consecutivos são representados por um único 零.
